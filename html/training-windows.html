디펜스 트레이닝

Windows Bug Hunting and Exploit Course를 통해 윈도우즈 환경에서 취약점을 찾는 방법과 익스플로잇을 작성할 수 있는 기술을 배울 수 있습니다. 윈도우즈 퍼징 방법부터 브라우저/커널 익스플로잇, 로직버그까지 윈도우즈 취약점의 거의 전 분야를 다루고 있습니다. 각 이론 설명마다 수강자분들의 이해를 돕기 위한 실습과정이 포함되어 있습니다.

등록

등록 페이지 링크

일정

  • 1일차 - Windows Fuzzing
    • Coverage Guided Fuzzer 구현 방법 및 운용 (WinAfl 내부 구조 이해, 운용방법 및 리얼월드 소프트웨어 대상 실습)
    • Grammar based Fuzzing (Domato 내부 구조 이해, 룰 작성 법, 운용방법 및 리얼월드 소프트웨어 대상 실습)
    • Script Engine Fuzzing (vbscript/jscript/chackracore 엔진 퍼징 실습)
  • 2일차 - Windows App Exploitation
    • 취약점 타입 및 Mitigation 소개
    • 취약점 타입별 1day 분석 및 익스플로잇 기술 소개
    • 주요 애플리케이션 익스플로잇 실습
      • 웹브라우저(IE)
      • PDF 뷰어(AdobeReader)
  • 3일차 - Windows Kernel Exploitation
    • 윈도우즈 커널 인터널 소개
    • 윈도우즈 커널 드라이버 핸들러 분석 방법
    • 윈도우즈 커널 드라이버 취약점 익스플로잇 실습
    • 주요 취약점 벡터(win32k) 동작 이해
    • 커널 1day(로컬 권한상승) 익스플로잇 분석 및 실습
  • 4일차 - Windows Logic Bug Hunting
    • 윈도우즈 시큐리티 기본 개념(사용자/권한 등) 소개
    • 권한상승 취약점을 찾기 위한 프로세스 소개
    • Sysinternals,RPCView 등의 도구 활용법
    • Com/Rpc 등 주요 취약점 인터페이스 액세스 방법
    • 권한상승 로직 버그 1day 분석
    • 권한상승 로직 버그 실습
